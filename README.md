# AI-Based Adaptive Dual-Mode Assistive Computer Control System

## Overview

The AI-Based Adaptive Dual-Mode Assistive Computer Control System is an AI-powered accessibility solution that enables touchless computer interaction using Hand Gestures and Head Movements.

The system allows users to perform common computer operations such as cursor movement, clicking, scrolling, screenshot capture, and mode switching without relying on a traditional mouse or keyboard. This project aims to improve accessibility and provide a more intuitive Human-Computer Interaction (HCI) experience.

---

## Features

### Hand Gesture Control

* Cursor Movement
* Left Click
* Right Click
* Scrolling
* Screenshot Capture
* Virtual Keyboard Access

### Head Movement Control

* Cursor Navigation Using Head Tracking
* Blink-Based Clicking
* Head-Tilt Scrolling

### Adaptive Mode Switching

* Seamless Switching Between Hand and Head Control Modes
* Real-Time Performance Using a Standard Webcam
